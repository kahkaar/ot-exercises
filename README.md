# ot-exercises

This repository will contain the project files and the weekly exercises for the _Intermediate Studies Project: Software Development Methods_ [`TKT20018`](https://studies.helsinki.fi/courses/course-unit/otm-fc35db8b-596c-4287-a03c-047e81e1254b/TKT20018?cpId=hy-lv-76) course at the **University of Helsinki**, as described in the [course material](https://ohjelmistotekniikka-hy.github.io/).

## project: db-utils

An utility application for inspecting local SQLite databases and exporting table and query results to `.csv` and `.json` files.
